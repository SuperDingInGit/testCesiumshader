# 🌊 Cesium 三维真实感河流仿真

基于 CesiumJS 和自定义 GLSL 着色器的实时三维河流仿真，模拟长江三峡（重庆→宜昌）段。

## 特性

- **三维水面法线扰动** — 利用 Cesium `tangentToEyeMatrix` 实现真实凹凸
- **菲涅尔反射** — 视线角度驱动的天空反射，低角度更强
- **太阳高光** — 响应 Cesium 场景光照的镜面反射
- **多层水流** — FBM 分形布朗运动 + 4层噪声叠加
- **动态泡沫** — 河岸边缘 + 湍流处自动生成
- **实时可调参数** — 流速、湍流、波浪、菲涅尔、高光等

## 快速开始

```bash
# 直接下载 index.html，用浏览器打开即可
open index.html
```

或者部署到任意静态服务器：

```bash
python3 -m http.server 8000
# 打开 http://localhost:8000
```

## 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | Cesium WebGL 3D 河流仿真（需要 WebGL 支持） |
| `canvas-demo.html` | Canvas 2D 版本（无需 WebGL，任何浏览器都能跑） |

## 操作

- **飞到三峡** — 俯瞰长江三峡全景
- **低角度观察** — 飞近水面，观察三维波纹和高光
- **俯瞰视角** — 正上方俯视
- 左侧面板可实时调节所有着色器参数
